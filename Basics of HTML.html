The Basics of HTML websites!




What is HTML?



To start, HTML stands for "Hyper Text Markup Language"

This is because its the "markup language" used to make Webpages; so like, its a language webpages understand that tell to do a specific action.

HTML consist of elements. Signals that tell the Webpage to display specific information or looks

For instance, if I wanted add text saying "hello world", I'd need to use: "<p>hello world</p>"

The less than and greater than signs are used as the signals to the page while the "p" means the text needs to be inserted as paragraph text

Moving forward, the "p" within that signal can be changed for another figure for a different meaning

Different types of HTML elements



In the context of the example shown above, there's a multitude of different kinds of elements that replace the "p":

  • <h1> - sets text selected as a header. "The Basics of HTML websites!" is an h1 element. The "1" can be 1-6 which make text smaller the higher it is.

  • <img> - inserts an image. these require more info such as image file, alternative text if it fails to load, and size. <img src="image file" alt="alternative name" width="###" height="###">

  • <a> - inserts a clickable link. They also require more such as its link, and the name it will display over it. <a href="link">display name</a>

  • <div> - these elements do completely nothing ... on its own. These select a multitude of HTML elements and require CSS elements to effect them all at once.
