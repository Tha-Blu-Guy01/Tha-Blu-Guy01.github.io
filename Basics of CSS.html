The Basics of CSS Style Sheets!




What is CSS?



CSS stands for "Cascading Style Sheet".

This describes the way certain decorating elements are displayed.

so if I wanted the text "Hi there" to be green, I'd add it as: "<p style="color:green">Hi there</p>".

Firstly, this is one of the many ways of adding a CSS element(s).

the "style=" shows that there's something you'd want to decorate in the text with "'color:green'" signifying what the change is.

Ways on using CSS



A way of using CSS elements can linked from another file document or a css doc; hence, "Style Sheet". This is called "External CSS"

Most pages in this entire website are formatted the same, they share similar css documents while this page has its own.

unlike the example, this is formatted differently as insteaad of "style=" you'd insert what HTML elements you want to affect. The "<" and ">" are not needed.
and instead of quotation marks, you'd add curly brackets surrounding the decoration commands.

an example being: p {color:green}

You can also decorate HTML elements on the same document in two ways: Internal CSS and Inline CSS

With internal CSS, this involves adding CSS elements into the "head" segment; plus, they are added in the same format as External CSS.
This will effect all HTML elements on the document that follow the element you inserted.

Finally, Inline is more direct as these are added only specifically to one HTML element

They are inserted within the HTML element as shown within the example.

These will only affect the HTML element they are with.

The CSS Hierarchy



When it comes to Webpages, you'll bound to have more than one CSS decoration in your document.

So how will these different forms of CSS interact with each other? Answer: they're in a hierarchy.

Within this hierarchy of deciding how a certain HTML element will be shown is dependent on who's closer to the element within the code. The code will be read top to bottom and apply in the order read.

  1. Inline CSS has the most power in all as they are right next to the element; thus, the HTML element will be shown with what the Inline says.

  2. Internal CSS is the second in the most power as they are also within the document but are held at the top.

  3. Finally, External CSS is at the very bottom in power as they are the furthest away from the HTML element being in a whole seprate document.
